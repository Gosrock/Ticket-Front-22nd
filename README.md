## 5. 폴더 구조

```bash
├── .github # 액션 워크플로우 세팅
├── .storybook # 스토리북 세팅
├── public
├── src
│   ├── apis # api 관련 파일 (에러 핸들링, axios 설정, 타입)
│   ├── assets # 에셋 파일
│   ├── components # 컴포넌트 파일
│   ├── hooks # 커스텀 훅
│   │   └── queries # 리액트 쿼리 관련
│   ├── pages
│   ├── stores # 리코일 관련 파일
│   ├── styles # GlobalStyle, ThemeProvider
│   ├── utils
│   ├── App.tsx
│   └── index.tsx
└── # configs
```
